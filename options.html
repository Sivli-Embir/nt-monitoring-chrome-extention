The above words are regex based searches. Each new line will be a separate search.

  • Post text will use /^word/i (case insesitive)
  • Tags will use /^#word/i (hashed case insesitive)

Warning: Currently nT Monitoring is not emitting any (known) events or hooks. As such we have to use a timer (see Seconds per check). If you are getting bad performace you can increase the seconds per check. You can also decrease it to less then a second (e.g. 0.5), this is not recommended or really needed.
